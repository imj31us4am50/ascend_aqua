
# Ascend Aqua Project
![Logo](https://imj31us4am50.github.io/imgs/ascendaqualogo.png)

Ascend Aqua is an educational platform focused on aquatic life, designed to help users learn interesting and engaging information about ocean animals and other marine creatures.


## Features

- Provides informative articles about marine life and ocean animals 🌊

#### Planned Features:

- Implementation of a posting system for user content
- Search bar for easier navigation and content discovery
- Mini-games section for interactive learning
- Account management system for user profiles and personalization
